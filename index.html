What is Data-Centric ML?

The Data-centric AI Competition aims to elevate data-centric approaches to improving machine learning model performance. In most machine learning competitions, a user is given a fixed set of data and asked to improve on the model to produce quality predictions. However in industry, the time machine learning practitioners spend improving data vastly outweighs the time spent improving models. The Data-centric AI Competition inverts the traditional format and instead asks the user to improve on the data given a fixed model. We will provide you with a dataset to which you can apply data-centric techniques such as fixing incorrect labels, adding data for side case tuning, apply data augmentation techniques, etc. You will then submit your altered dataset for evaluation. Participants have until September 4th, 2021 to submit. The top 3 winners will be invited to a private roundtable event with Andrew Ng to share ideas and explore how to grow the data-centric movement, and will have their work highlighted on The Batch and other DeepLearning.AI and Landing AI channels. Good luck!
